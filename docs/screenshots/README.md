# AuditChain Screenshots Guide 📸

To keep the project's visual presentation high-quality, please capture screenshots following this guide.

## Recommended Settings
- **Resolution**: ~1600px wide
- **Theme**: Light Mode (standard shadcn/ui neutral theme)
- **Format**: `.png` (lossless)

## Required Screenshots

1. **`dashboard.png`**
   - **Page**: `/` (Home)
   - **Content**: Show the hero section, the KPI cards, the risk analysis chart, and the recent audits table.

2. **`live-audit.png`**
   - **Page**: `/audits/[run_id]` (Running)
   - **Content**: Capture while an audit is in the middle of a phase (e.g., Reconciler). Ensure the pipeline visualizer at the top is visible and the "Current Phase" panel shows streaming activity.

3. **`report.png`**
   - **Page**: `/audits/[run_id]` (Completed)
   - **Content**: Scroll to show the final report header, the risk score gauge, and the Executive Summary section.

4. **`add-company.png`**
   - **Page**: `/` (Home)
   - **Action**: Open the "Add Company" modal and type "Microsoft" or "Apple".
   - **Content**: Capture the modal with the search suggestions list visible.

## Placement
Save all files directly into this directory (`docs/screenshots/`).
